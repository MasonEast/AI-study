比如：判断肿瘤是否良性，垃圾邮件过滤

#### 逻辑回归

逻辑回归是一种分类算法，它试图预测一个离散的值，例如 0 或 1，是或否，真或假。
逻辑回归的输出是介于 0 和 1 之间的值，这被称为逻辑函数或 Sigmoid 函数。逻辑回归试图将输入数据映射到 0 和 1 之间，这被称为分类。
