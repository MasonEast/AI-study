## 监督学习

输入到输出价值

关键特征：提供学习案例，每个案例都有正确的答案。给定输入 x 的正确标签 y，学习算法的目标是找到函数 f，使得 f(x) 接近 y。

主要算法有：线性回归，分类算法

### 线性回归

从无数的数值中，找到一条直线（曲线），使得这条线与所有点的距离之和最小，从而预测一个最接近的数值。

### 分类算法

与回归不同，这里可能只预测一小部分可能的输出，比如 0 或 1，或者一个类别。回归试图预测无限多个可能数字中的一个
